文件的切割和合并

2016/6/21 posted in  Java

2016年06月21日

问题

需要一个能把文件切割成指定大小和合并起来的程序

思路

切割

  1. 创建一个输入流读取要切割的文件
  2. 定义切割的大小和存放路径
  3. 用输出流切割文件并命名
  4. 用配置文件记录下切割信息
s=>start: 开始切
e=>end: 切割完 
c=>condition: 判断是否存在此目录
o=>operation: 创建目录
op=>operation: 读取要切割的文件,定义切割大小和存放路径
oper=>operation: 切割文件并命名并记录切割信息
s->op->c(yes)->oper->e
c(no)->o->oper

合并

  1. 读取要合并文件目录的配置文件
  2. 读取要合并的文件
  3. 创建集合装载流对象
  4. 枚举集合
  5. 创建序列流装载枚举
  6. 用输出流遍历序列流并写出数据
s=>start: 开始合并
e=>end: 合并成功
e1=>end: 合并失败
c=>condition: 判断是否存在配置文件
o=>operation: 读取配置文件
co=>condition: 判断合并文件数量和记录是否一致
op=>operation: 读取碎片，合并文件
s->c(yes)->o->co(yes)->op->e
c(no)->e1
co(no)->e1

代码

切割

import java.io.File;
import java.io.FileInputStream;
import java.io.FileOutputStream;
import java.io.IOException;
import java.util.Properties;

public class Java20160621_2 {
        //指定切割大小
    private static final int SIZE =1024*1024 ;

    public static void main(String[] args) throws IOException {
            //指定切割文件
        File file = new File("Cannie - 遥远的歌.mp3");
        FileInputStream fis = new FileInputStream(file);
        byte[] buf=new byte[SIZE];
        int len=0;
        int count=1;
        //指定存放目录
        File dir=new File("/test");
        if (!dir.exists()){
            dir.mkdir();
        }
        FileOutputStream fos=null;
        //按照切割大小开切
        while ((len=fis.read(buf))!=-1){
            fos=new FileOutputStream(new File(dir,(count++)+".part"));
            fos.write(buf,0,len);
            fos.close();
        }
        //创建配置文件记录信息
        Properties prop=new Properties();
        FileOutputStream fis2=new FileOutputStream(new File(dir,count+".properties"));
        prop.setProperty("name",file.getName());
        prop.setProperty("partcount",count-1+"");
        prop.store(fis2,"file count info");



        fos.close();
        fis.close();
    }
}

合并

import java.io.*;
import java.util.ArrayList;
import java.util.Collections;
import java.util.Enumeration;
import java.util.Properties;

public class Java20160621_3 {
    public static void main(String[]args) throws IOException {

        //获取指定目录下的配置文件对象
        File dir=new File("/test");
        File[] files=dir.listFiles(new SuffixFilter(".properties"));
        if (files.length!=1){
            throw new RuntimeException("我总感觉文件数目不对,要么不存在,要么数量太多");
        }

        //记录配置文件对象
        File confie=files[0];

        //获取该文件中的信息
        Properties prop=new Properties();
        FileInputStream fis=new FileInputStream(confie);
        prop.load(fis);
        String name=prop.getProperty("name");
        int partcount=Integer.parseInt(prop.getProperty("partcount"));

        //获取该目录下的碎片文件
        File[] parts=dir.listFiles(new SuffixFilter(".part"));
        if (parts.length!=partcount){
            throw new RuntimeException("碎片数量不对劲");
        }

        //将碎片文件封装和流对象管理并存储到集合中
        ArrayList<FileInputStream> al=new ArrayList<FileInputStream>();
        FileInputStream f=null;
        for(int i=1;i<=partcount;i++){
            f=new FileInputStream(new File(dir,i+".part"));
            al.add(f);
//            f.close();
        }

        //将多个流合并成一个序列流
        Enumeration en= Collections.enumeration(al);
        SequenceInputStream sis=new SequenceInputStream(en);

        //读写操作
        FileOutputStream fos=new FileOutputStream(new File(dir,name));
        int len=0;
        byte[] b=new byte[1024];
        while ((len=sis.read(b))!=-1){
            fos.write(b,0,len);
        }

        sis.close();
        fos.close();
    }
}
