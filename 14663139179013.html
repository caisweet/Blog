删除多级目录

2016/6/19 posted in  Java

2016年06月19日

问题

Java中不支持直接删除不为空的多级目录，所以要解决这个问题要曲线救国。

思路

  1. 创建一个File对象，内容是一个目录地址
  2. 创建一个方法，专门用来遍历File对象数组，看是不是目录
  3. 是目录的话，用递归调用方法继续遍历
  4. 是文件的话，删除
  5. 最后文件都删除之后，删除目录
  6. 可以调用输出语句来判断有没有删除成功
st=>start: 创建一个File对象
e=>end: 结束
op=>operation: 遍历
opt=>operation: 删除文件 
cond=>condition: 是否目录？
delete=>operation: 删除目录
st->op->cond
cond(yes)->op
cond(no)->opt
opt->delete->e

代码

import java.io.File;

public class Java20160619 {
    public static void main(String[]args){
        File dir = new File("/test");
        deleteIt(dir);
    }
    public static void deleteIt(File dir) {
        File[] files=dir.listFiles();
      //遍历目录，如果遍历到的是目录，递归调用自身继续向下级目录遍历
        for (File file:files) {
            if (file.isDirectory()){
                deleteIt(file);
                //如果遍历到的是文件直接删除
            }else{
               System.out.println(file+":"+file.delete());
            }
        }
        //遍历结束后删除空目录
        System.out.println(dir+":"+dir.delete());
    }
}
