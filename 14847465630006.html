至于公开，如果只是自己看，那就只是总结。而公开，只要能给别人带来一点益处，那就值得。文字排版，采用的是Markdown，顺便也推荐一下

概括

16年换了笔记本和手机，算作是加入苹果生态系统的开始。由于工具的便利，所以养成了一些一直以来很想养成的习惯。以前有尝试过，都因为懒失败了。

备份

备份的重要性大家都知道，只是因为不知道怎么做。（我是因为懒）

通讯录

  1. iOS
    我主要推荐两种方式,icloud自带备份通讯录(推荐)、QQ通讯录助手(备用)

  2. Android
    由于不同厂商自带的云服务不同，所以还是推荐QQ通讯录助手。主要是以后如果换成其它厂家的Android手机也方便转换。(其他可以备份通讯录的软件也可以，主要是大厂商，可以信任的即可)

照片

  1. iOS
    iCloud备份，优点是自动化，如果你有几个苹果设备的话，你会喜欢这种便利的，缺点就是免费容量只有5G，再加要花钱每月订阅。
    Google Photo备份，优点是标清照片不限容量，缺点是得准备梯子。

  2. Android
    Google Photo备份，优缺点如上。
    其实Android上的选择很少，毕竟我不太敢把自己的照片存在国内手机服务商的云上。

  3. 总结
    不管你照片存在哪里，主要考虑一下几个原则。

    1. 安全性

    2. 便捷性

      第一点很好理解，我想仔细讲讲第二点，因为亲身经历过。在2016年以前，我都是把照片存在QQ空间相册里面。一开始没感觉到什么，顶多是上传费点时间。直到我想所有照片取出来的时候，我发现坑爹的QQ空间相册没有导出照片的功能！也就是说你把照片上传到QQ空间相册后，想取回来，只能一张一张保存下来！还有，空间相册删除的照片，据我观察只是在数据库给照片加个标识删除，没有真正的从数据库中删除掉，因为在去年QQ空间的一个漏洞中，发现以前删掉的照片还在回收站中(默认QQ空间相册删除的照片30天后从回收站中自动删除，当然也可以手动提前)。自此QQ空间相册一生黑。

      当然如果你是苹果系统的，那当然iCloud几乎就是唯一也是最好的选择了。（推荐给Apple ID设置两步验证）

      由于15年开始喜欢拍照，日积月累，已经在iCloud中存了13000+张照片，100+个视频，当然也订阅了50G的iCloud空间。可以预知的是，这些照片将会陪伴着我一生了。

      如果是Android，推荐Google Photo，但是需要自备梯子。那说实在，不如压缩起来存到电脑里面，虽说便捷性没有了，但是安全性提高了。还有，切记异地容灾备份（别只存在一个设备或位置啊！）

生日

在17年以前，我都是通过软件来备份生日的，这样有个缺点，就是如果过我想看生日只能进入软件里面看。（虽然有通知栏和提醒，但毕竟是装多了一个软件，对于我这种能简单就不想麻烦的人自然是不爽）转入苹果生态系统之后，发现联系人（也就是通讯录）里面有一向信息专门可以用来填生日（更可贵的是支持农历）。所以就义无反顾的从软件迁移到联系人里面，而且我发现可以在原生日历里面显示出来，也支持搜索功能。

微信聊天记录

相信现在大部分人都使用微信聊天，所以微信聊天记录对你有时候比较重要？如果手机之间的转移的话，微信本身就支持这个功能，只是入口比较深。

今天微信MAC内测版更新到2.2.0，支持手机聊天记录备份到电脑，和从电脑恢复聊天记录，不过微信windows版有没有这个功能我不知道啊。

习惯

写日记

这是我很喜欢的一个习惯，在以前，也尝试过，只是常常就中断了。直到去年换笔记本之后剁手入了个日记软件，然后从2016年4月3日记日记到现在。我很在乎每个日子里的情感和所想，更别提经历过的那些有趣的事情。再加上我记性差，很容易可是又很害怕忘记以前的事情。

  1. 自拍
    2016年4月28日开始，每天写日记都会在最上面放一张自己当天的照片。也是从这个之后，开始有意识且喜欢上和朋友自拍，因为可以放在日记里面。

  2. 总结
    我觉得记日记最难的是开头坚持，只要坚持超过一个月以后就很自然了。我推荐的做法是，找一个自己喜欢的日记软件(当然如果你喜欢写字用日记本也行，只是我觉得电子化相对于我比较容易保存和备份)，设置提醒，从今天开始，从现在开始，写日记！不想写的时候就写一句今天不想写也行。(如果是Mac和iPhone上的写日记需求，我推荐Day One)

记账

  1. 这在以前是我很喜欢的一个习惯，因为作为一个月光族，每次都不知道钱花到哪里去会有一种无力感。以前时候记不了账是因为懒和怕麻烦。

  2. 16年换笔记本和手机之后，我决心卷土重来，尝试养成记账的习惯。又是一样的开始，我剁手了一个记账软件。。。。(这个我实在是推荐电子化，不推荐手记账本了。我相信带手机在身上的概率大过带账本在身上。)

  3. 养成习惯的重点在于理清思路，先分清楚支付账户(银行卡、现金、羊城通)，信用卡(泛指一切可预支的账户，如花呗、借呗、白条、信用卡)，资产(租房押金、共享单车押金、就是可以换回来钱的资产！)，然后每次一消费就记下来，至少每一星期要进行一次对账，不然很快就会发现对不上账本了！

  4. 总结
    例如我一月份到现在的交通费用，很直接明了。

日历安排

16年底对日历有了新的理解(除了上文提到的记生日作用),还有对时间的安排，一如既往的我记性很差，如果我约了别人在未来做什么事情，如果我不记到日历中，我对遗忘很有自信。另一方面，我很向往有计划性的生活，所以我的日历粗略分成生活、亲朋、工作，来给不同的事件安排归类，需要提醒的还可以设置，写来感觉还是贴图片来得直接些。

事件安排

习惯于给要做的事情而且对时间不敏感的类型归入事件安排(日历对我来说更像是时间安排)。

如图对想看的电影进行任务化，防止遗忘，且观看完后可以打勾确认完成。当然，更多功能还要靠自己摸索。

流量

  1. 概述

    如果你也是流量大户，你会知道我为什么要写这个话题。在以前，曾经因为使用流量超出多交了很多流量费。所以我推荐一个解决方式，使用流量日租卡。特别是你的手机有两个卡槽的话，或者有两个手机？

  2. 联通流量日租卡
    1元500m，一开始我也是使用联通卡，虽然发现有所下降但还是很高。

  3. 电信流量日租卡
    1元800m，后来我就换成电信到现在了。

  4. 总结
    流量日租卡的优缺点都很明显，优点：流量日结，不用不扣费，月租很低。缺点：只能在归属地优惠，也就是说，如果你买了一个广州号码的流量日租卡，就只能在广州上网优惠。

密码管理

  1. 概述

    在15年和更早以前，我都是大部分网站同时使用一个密码。也因此如果一个网站密码泄露那么大部分网站的账号基本就废了。但是当时有这样一个纠结点就是如果我所有网站密码都设成不一样我又记不住。

  2. LastPass
    16年开始尝试使用这个服务，订阅制，一年好像是90块。优点是很轻，宣称很多加密，特点是云端化，就是密码库是加密后存在服务器中，这个算缺点？

  3. 1Password
    16年后期转到1Password，首先功能比LastPass多了记录二步验证的功能，不知道现在有没有跟进。优点是密码库本地化，而且各种平台使用起来都很方便，特别是电脑端的小插件，可以快捷键填充密码。也是订阅制1个月27元，不过经常有优惠活动，我记得家庭版有优惠，在此，羡慕一下能组成家庭版的家庭。

  4. 总结
    综上所述，使用哪个软件取决自己的需求。当然你如果不是像我这样的注册网站狂魔，其实靠纸笔也能解决，主要是得要保存好！还有建议不要几个网站使用同一个密码！能开两步验证的要开！！

域名

  1. 概述
    说实话，16年注册购买域名的时候还没想好用它来做什么，所以一开始只想到了用来放网站。在此感谢一下GitHub Pages提供的部署服务，虽然只是静态网站，但已经很感谢了。那就说一下注册到现在的心得？

  2. 注册域名

    1. 阿里云 国内域名商的首选咯，优点是本地化很好，缺点就是你懂的。
    2. Godaddy 国外域名商的推荐，优点你懂的，缺点就是服务容易被墙，特别是DNS服务器。

  3. 部署

    1. 静态网站 首选就是GitHub Pages，无可黑，且不用备案，就是有时候服务会有点慢。
    2. 动态网站 国内推荐阿里云和腾讯云当下谁的优惠政策好，缺点是要备案域名啊。国外推荐Amazon Web Services，优点服务好，缺点贵。

  4. 总结
    也是到后来，我发现静态网站也可以做很多事情，虽说不可以和数据库交互。但是对于目前的我来说，静态页面就是你现在所看到的这个网站咯。以后还想研究一下能做什么。

网盘

  1. 概述

    其实iCloud也算是一种网盘，所以目前我在用网盘存东西。照片、视频、联系人、事无巨细的资料和文档都放在iCloud，而比较私人的，例如发票、密码库和网站备份都存在Dropbox，一个一个拿出来单独讲好了。

  2. iCloud
    苹果家的网盘，有点在于如果你苹果设备比较多，那么一体化的优点就特别明显了，例如照片流，照片会自动上传。也就是说手机拍了照，回到家接入wifi，电脑上也自动有了。所以我全部照片和视频都放在iCloud上。

  3. Dropbox
    各种软件存资料的首选，优点是安全，缺点是依旧需要自备梯子。而且免费空间较小，可以上淘宝买服务扩充一下空间。所以用来存比较小的，私人的资料。

  4. 百度云
    这个就呵呵了，首先把资料传上去就真的不是你个人的了，下载速度不是会员也慢！所以只推荐用来存电影等没了不会害怕的资料。

  5. 总结
    如果Android推荐微软的OneDrive,备选Google Drive(需自备梯子)。前者Windows系统内置，后者原生Android内置。

    总而言之，重要的资料请备份！

    梯子

    我推荐多态吧，虽然需要付费，但是禁不住服务好，客户端都有，支持几个设备同时使用，速度也不错，日常使用够了。

娱乐

到这部分好像干货就变少了，真的只是个人总结了。

电影

  1. 概括

    从15年开始，已经习惯大部分时间一个人去电影院看电影了，这也是我最主要的娱乐方式。那就分享一些自己看电影的心得？

  2. 选电影

    一个人看电影更注重的当然的是电影的质量，难不成你是去电影院睡觉？推荐首选参考豆瓣评分吧，参考的原则是评分高的不一定合你的口味，但是评分低的一定大部分人觉得不好看，(备选IMDB，更偏向外国人的口味)。

  3. 购票渠道

    推荐淘票票，找在距离与票价权衡之下最好的电影院，经常看电影的话直接上影城卡。推荐早上10点后看价格，经常有特惠活动。

  4. 私心

    很少推荐电影，但是这一部忍不住想推。就是今年2月14日上映的《La La Land》(《爱乐之城》)，大概是今年最不可不看的院线电影，好电影，可不要错过喔！至于为什么好看，不如先去看？

电视剧

这个话题众口难调诶，推荐还是参考豆瓣评分。当然每个人有各自的喜好，我就各自说一部印象深刻的咯。

  1. 韩剧
    直接上《请回答1988》，请回答系列一直保持良好的出品。

  2. 日剧
    《火花》，如果看得下去的话。

  3. 美剧
    《毒枭》，魔幻现实主义。

音乐

  1. 软件
    这段时间尝试过的Apple Music和Spotify，加上一直在使用的网易云音乐，以及以前使用过的QQ音乐。前两者更适合喜欢外语歌曲的用户，毕竟外国公司外国歌曲版权比较集中(Spotify需自备梯子)。后来者说仔细可以变成为何从QQ音乐迁移到网易云音乐，主要在于网易云音乐的UI好看，评论区加分。缺点当然是曲库(当然靠音乐云盘可以解决基本的需要)。QQ音乐当然要说缺点了，界面丑加广告多！

  2. 总结
    收藏和分类的习惯有助于随时可以找到想听的音乐。
