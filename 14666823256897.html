截取字节解码

2016/6/23 posted in  Java

2016年06月23日

问题

在GBK编码情况下，有一串字符串，字母混合汉字。由于汉字占两个字节，如果按字节数截取，截取刚刚好的数量，正常解码显示

思路

  1. 获取字符串的长度
  2. 设置要获取的字节长度
  3. 用字节数组存储字符串
  4. 判断截取的字节编码正负
  5. 如果正就直接解码输出
  6. 负数就往回遍历直到正数并计算个数
  7. 取模2是0就直接输出，有余数就-1输出

代码

import java.io.UnsupportedEncodingException;

public class Java20160623_1 {
    public static void main(String[]args) throws UnsupportedEncodingException {
        String str=new String("abc原来你也sdsd在这sds里s");
        //获取字节串长度
        int len=str.getBytes("GBK").length;
        for (int x=1;x<len;x++){
        System.out.println("截取"+x+"个字节,结果是"+getByte(str,x));
    }
    }

    private static String getByte(String str,int len) throws UnsupportedEncodingException {
        //创建数组存储字节串
        byte[] buf=str.getBytes("GBK");
        int count=0;
        //遍历要截取的字节窜
        for (int x=len-1;x>=0;x--){
            //大于0说明是单个字节
            if (buf[x]>0){
                break;
            }else{
                //否则说明是半个中文
                count++;
            }
        }
        if (count%2==0){
            //刚刚好字节输出
            return new String(buf,0,len,"GBK");
        }else{
            //否则倒退1位输出
            return new String(buf,0,len-1,"GBK");

        }
    }
}

总结

编码解码问题需要根据编码表的特性分析从而进行解决。
